# BA-kit Home

Đây là space `Home` cho GitBook site của BA-kit.

Mục đích của space này:

- làm landing page cho toàn bộ site
- giới thiệu ngắn BA-kit là gì
- điều hướng người đọc sang `Documentation` hoặc `Changelog`

## Nên đặt gì ở đây

- value proposition ngắn
- đối tượng sử dụng
- link sang documentation chính
- link sang changelog
- CTA như `Get Started`, `Read the Docs`, `View Updates`

## Điều hướng đề xuất

- `Documentation`: tài liệu sản phẩm và hướng dẫn sử dụng
- `Changelog`: lịch sử thay đổi và cập nhật phiên bản
