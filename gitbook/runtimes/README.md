# Runtimes

Section này chia hướng dẫn theo từng môi trường chạy:

- `Claude Code`: runtime có slash command style rõ nhất
- `Codex`: repo-native hoặc runtime install
- `Antigravity`: prompt-based, không có slash command

Nếu anh đang viết tài liệu GitBook cho end user, nên để mỗi runtime một page riêng thay vì trộn chung. Như vậy user vào đúng tool họ đang dùng và copy prompt trực tiếp.
