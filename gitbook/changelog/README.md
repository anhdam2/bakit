# BA-kit Changelog

Đây là space `Changelog` cho GitBook site của BA-kit.

Mục đích của space này:

- ghi lại các thay đổi quan trọng của BA-kit
- tách release notes ra khỏi documentation chính
- giúp user theo dõi thay đổi về workflow, templates, runtimes, và docs

## Nên đặt gì ở đây

- theo mốc version hoặc theo ngày
- breaking changes
- migration notes
- runtime-specific updates cho Claude Code, Codex, Antigravity

## Format đề xuất cho mỗi entry

- ngày phát hành
- version hoặc tag
- phần `Added`
- phần `Changed`
- phần `Fixed`
- phần `Migration notes` nếu có
